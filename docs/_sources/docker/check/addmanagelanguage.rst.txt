#####################
管理画面の日本語化
#####################

管理画面の日本語化の手続きをします。詳細な手順は、Sitecore のインストールを参照してください。ここでは手順の画面のみ記載しています。

.. image:: images/addmanagelang01.png
   :align: center
   :width: 400px
   :alt: ログイン

Desktop を開きます

.. image:: images/addmanagelang02.png
   :align: center
   :width: 400px
   :alt: Desktop を開く

右下の Master を Core に切り替えます

.. image:: images/addmanagelang03.png
   :align: center
   :alt: Core に切り替える

左下のスタートメニューから `Control Panel` を選択します。

.. image:: images/addmanagelang04.png
   :align: center
   :width: 400px
   :alt: Control Panel を開く

Localization メニューにある `Add a new language` をクリックします。

.. image:: images/addmanagelang05.png
   :align: center
   :width: 400px
   :alt: 言語の追加

`Japanese (Japan): 日本語（日本）` を選択して、言語の設定を追加します。

.. image:: images/addmanagelang06.png
   :align: center
   :width: 400px
   :alt: 言語の追加

Localization メニューにある `Import languages` をクリックします。

.. image:: images/addmanagelang05.png
   :align: center
   :width: 400px
   :alt: 言語リソースの追加

Sitecore のサイトからダウンロードした言語ファイルをアップロード、指定して、インポートを実行します。

.. image:: images/addmanagelang07.png
   :align: center
   :width: 400px
   :alt: ファイルの指定

.. image:: images/addmanagelang08.png
   :align: center
   :width: 400px
   :alt: インポート

インポートの作業が終わったあと、 `My Settings` にある `Region and language options` をクリックします。

.. image:: images/addmanagelang09.png
   :align: center
   :width: 400px
   :alt: 言語の設定を開く

表示する言語を日本語に設定します

.. image:: images/addmanagelang10.png
   :align: center
   :width: 400px
   :alt: 日本語に設定

左上にある Sitecore メニューのアイコンをクリックしてホームに戻ります。

.. image:: images/addmanagelang11.png
   :align: center
   :width: 400px
   :alt: 管理画面の言語が変わる

管理画面の言語が変わりました。

この手順の後、SXA の日本語リソースも適用してください。
