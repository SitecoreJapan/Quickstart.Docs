Sitecore Experience Accelerator¶

Sitecore Experience Acceleartor は Sitecore の環境に Web サイトを立ち上げるために提供されているコンポーネント集となります。コンポーネントを表示する方法を複数設定したり、拡張することができるため、サイト構築で非常に有用なコンポーネントとなります。ここでは、基本的な手順を紹介しています。
